
# Vi-Sakha

Vi-Sakha is a personalized AI assistant built on a **Retrieval-Augmented Generation (RAG)** pipeline.  
Unlike traditional chatbots that rely solely on language models, Vi-Sakha prioritizes **trusted knowledge sources** to deliver accurate, context-aware, and reliable answers.

##  Key Features
- Smart answer prioritization ensures verified responses first  
- Multilingual support via translation APIs  
- Modern chat interface for easy interaction  
- Knowledge-first responses reduce hallucinations  
- Stored conversations improve the RAG Database over time  
- Admins can upload/update questions without retraining  
- Real-time instant response delivery  
- External tool integration through MCP

 ## 🛠️ Technologies Used

### Frontend
- React  
- TypeScript  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB  
- Chroma / Vector DB  

### Language models
- DeepSeek
- GPT-OSS
- Qwen
- Ollama models    

### Authentication & Integrations
- Firebase for secure login and access control  
- Sarvam AI API (or similar) for translation and multilingual support  
- MCP (Model Context Protocol) servers for structured external data integration
##  Project Goals

- Deliver accurate, context-aware answers by prioritizing trusted knowledge sources.
- Provide a reliable assistant for academic, official, and support use cases.  
- Enable multilingual interaction so users can query in any language.  
- Offer a modern, user-friendly chat interface for seamless engagement.
